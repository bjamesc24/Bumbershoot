Bumbershoot Festival Companion

React Native (Expo) mobile application.

# How to Run the Mobile App #
1️⃣ Install Dependencies

After cloning or downloading the project:

1) cd app  
2) npm install


This step is required.

The node_modules folder is NOT included in the repository and must be generated locally.

2️⃣ Start the Application
npm start

This will start Expo and display a QR code in the terminal.

3️⃣ Run on a Phone (Recommended)

Install Expo Go from the App Store (iOS) or Google Play (Android).

Make sure your phone and computer are connected to the same Wi-Fi network.

Scan the QR code shown in the terminal.

If the app does not open:

a) Press t in the terminal

b) Switch the connection mode to Tunnel

c) Scan the new QR code

⚠️ Important Notes

Do NOT commit node_modules

Do NOT commit .expo

Always run npm install after pulling new changes
